# breakblog-vue

本项目使用：vue2.6 + element-ui 技术实现个人博客后台管理，可视化编辑器采用 tinymce 。  
具体博客项目请看 [braekblog](https://github.com/tw-huang/breakblog) 。

## 使用说明

1. 安装依赖 `yarn install / npm install`
2. 运行项目 `yarn serve / npm run serve` 
3. 浏览器打开控制台输出的地址
